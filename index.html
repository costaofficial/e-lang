What is E

E is a general-purpose language. You describe when to do something, on what, and what to do. The runtime handles the complexity.

Declarative. Human-readable. Event-oriented.

◆ ◇ ◆ ◇ ◆
Example
// every day at 2am — export & notify
time every day at 02:00 do
    with browser do
        open "https://connect.api"
        with page { timeout: 10s } do
            login "user" "pass" or stop
            click "#export-all"
            wait download
        done
    done
done or log error
Features

⏰ Time

Schedule scripts with time every day at — cron-like scheduler underneath.

🌐 Browser

open, find, click, wait — powered by Playwright.

🔄 Retry

retry N times do ... done with automatic backoff.

⚠️ Fallback

Every statement can have an or fallback — local or global.

📁 System

write, create, run — filesystem and shell.

📧 Email

email to "addr" file "x" — SMTP ready.

Try it
$ git clone https://github.com/costaofficial/e-lang.git
$ cd e-lang
$ python3 runtime/run_e.py examples/hello.e

# with a real browser
$ pip install playwright
$ playwright install chromium
$ python3 runtime/run_e.py --live examples/browser_demo.e
